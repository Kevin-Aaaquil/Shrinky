# Shrinky Url

## this is a initial code for URL shortner

## POST : send actual url
```
POST http://localhost:/create/link
{
    "link":"www.example.com"
}
```

## GET : enter shortend URL
```
GET http://localhost:3000/{shortened-code}
```
